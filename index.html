What are observables?

Collection of data delivered over time

var data = [1, 2, 3, 4, 5, ...];

Observable

Passes data to observer

  • next()
  • error()
  • complete()

Observer

Handles data of the observable

  • subscribe(next, error, complete)
  • unsubscribe()
